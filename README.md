# Informatics Engineering Ontology (KMS)

Este repositorio contiene la ontología desarrollada como parte de la tesis: **"Diseño e implementación de un sistema de gestión de conocimiento (KMS) basado en grafos de conocimiento para la navegación y descubrimiento de recursos de Ingeniería Informática"**.

## 🗂 Estructura del Repositorio

- **`/src`**: Contiene el código fuente de la ontología en formato Turtle (`ontologia_informatica.ttl`). Aquí residen tanto la T-Box (estructura y axiomas lógicos) como la A-Box (instancias).
- **`/docs`**: Contiene la documentación estática generada mediante OWLDoc.

## 🌐 Documentación (OWLDoc)

La documentación técnica de las clases, jerarquías y propiedades de objeto/datos está alojada mediante GitHub Pages.

👉 **[Ver documentación de la ontología](https://giano-montano.github.io/informatics-engineering-ontology/)**

## 🛠 Tecnologías y Herramientas

El modelado formal de este proyecto se sostiene sobre las siguientes tecnologías:

- **Lenguaje:** OWL 2 / sintaxis Turtle.
- **Edición de Ontologías:** Protégé.
- **Motor de Inferencia Lógica:** HermiT.
